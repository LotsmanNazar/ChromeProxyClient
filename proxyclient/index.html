Как это работает

ProxyClient - это проксий клиент, который позволяет вам хранить ваши прокси. Сам по себе клиент не предоставляет прокси сервера. Вам необходимо самостоятельно найти или создать прокси сервер, а затем добавить его в спиоск.

Далее вы сможете использовать выбранный из списка прокси. Убедитесь, что другие расширения для прокси/vpn отключены в вашем браузере.

FAQ

Где найти прокси сервер?

Поиск прокси сервера

Вы можете найти прокси сервер в интернете. Существуют сайты аггрегаторы, на которых размещены данные для подключения.

 Какой тип прокси мне нужен (SOCKS, HTTP)?

HTTP прокси оптимизирован для веб-трафика, в то время как SOCKS прокси универсален и подходит для любого типа интернет-соединения.
